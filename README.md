# modelwork2

A Zig-based numerical and gradient computation framework.

## Project Layout

- `src/` - core framework modules
- `tests/` - automated test suite
- `examples/` - runnable examples and demos

## Build

```bash
zig build
```

## Test

```bash
zig build test
```

## Run Examples

```bash
zig build run
```

## Generate Docs

```bash
zig build docs
```

## Versioning

This project follows Semantic Versioning. See `CHANGELOG.md` for release history.
